Project Manager app is a one user type software, which means that it has no self-registration form. In the installation, the database should be configured to have an account for this to be used.

1. Log in

This is the first screen you will see. Assuming you have configured the database, just type in your credentials and once valid you can now access the menu screen.

2. Menu and Job Orders

After Login, you will be redirected to the main menu of the application. which leads you to various functinalites such as Projects, Customers, and Employees . You can also see the finished job orders by clicking on the hyperlink 'See finished job orders' .

This also serves as the main screen for the job orders, which shows an overview on what jobs are in progress. Below the 'Employees' button, the Print button allows you to export the table for Job Orders in Excel format for printing.

On the right hand side of the screen shows the form to add new job orders. After selecting and typing all necessary information, click add to confirm.

The lower right side of the screen is where you can see the backup button to backup all data and structure of your database

3. Projects

This screen shows the project details, and adding new project can be accessed on the right hand side. This is where you can assign employees to new projects.

There is also a green CSV icon on top of the form the allows users to export data into a CSV file for the Employees table and the table can also be printed using the print button which imports the data to a formatted excel workbook.

4. Customers

Generally, all the main screens have the same functionalities. The Customer screen shows the customer profiles and lets you add new customers. With the Customer Project button, you cann access the view that shows the customer and their project.

This also has a function in which you can import data from csv into your datagrid, you just have to add it the file to the directory with the Add CSV file to directory. Then after adding it, load the added file into your datagrid by selecting it.

5. Employee

This screen shows the employee informations, including passwords.This screen allows you to add employees. Passwords and Usernames can be nullable so that some employes can have profles but no way to have access into the system.
